## ITM-TICKETS
ITM-Tickets es una plataforma basada en microservicios en .NET 8 que permite simular la compra de entradas para el concierto de fin de semestre del ITM.

El sistema está diseñado para soportar múltiples usuarios comprando al mismo tiempo y evitar problemas como:
- sobreventa de sillas
- fallos en pagos que dejen datos inconsistentes
- caídas del sistema ante muchas solicitudes

Para esto se utilizan conceptos de programación distribuida, resiliencia y el patrón SAGA.
